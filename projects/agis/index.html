AGIS: Autonomous Greenhouse Integrated System

Goal: Make greenhouses the next household appliance for sustainable food production.

  • Developing an embedded C++ HAL and API to lower entry barriers for agricultural automation.
  • Using Computer Vision to detect mycelium contamination and monitor plant health.
  • WebApp + XR Integration with Digital Twin for remote monitoring.
  • Open-source framework for community-driven research in sustainable agriculture.
